# fire-shop-lite
fire-shop-lite是一个基于“api工厂”（以下简称工厂）后台的电商类微信小程序，前端代码完全开源。因工厂后台的基础api是免费的，因此，结合fire-shop-lite，您可以轻松搭建一套微信小程序电商系统。我们提供尽可能详细的操作手册，即使不是开发人员，只要按照手册傻瓜式操作，也能快速搭建一套属于自己的电商小程序。帮助您以最低的成本试水微信电商。
